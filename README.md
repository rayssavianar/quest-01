💻 Projeto: Huddle Landing Page - Quest de Responsividade
Durante esta quest, desenvolvi uma landing page responsiva baseada no layout da Huddle, com foco na prática de display: flex e media queries.

Neste desafio, trabalhei:

📱 Responsividade completa para dispositivos mobile e desktop

🎯 Estruturação com Flexbox para alinhar e distribuir os elementos com precisão

🎨 Adaptação do layout usando @media para diferentes larguras de tela

🌐 Aplicação de background com imagem e cor de fallback

🧩 Organização semântica do HTML e estilização eficiente com CSS puro

O objetivo foi criar uma página visualmente agradável, funcional em todas as resoluções e com um código limpo e bem estruturado. A experiência reforçou meus conhecimentos em design responsivo e na construção de interfaces modernas com CSS Flexbox.
